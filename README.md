# Tailwind-website

Tailwind Portfolio Website
Project Brief

This project was developed as part of my journey learning Tailwind CSS and modern front-end web development. The goal was to recreate a high-fidelity portfolio website from provided wireframes while applying responsive design principles, utility-first styling, Flexbox, and CSS Grid layouts.

The website consists of multiple interconnected pages that work together as a complete portfolio experience:

'[tailwindcss-index.html](https://github.com/user-attachments/files/28724736/tailwindcss-index.html) - Home Page
'[tailwindcss-portfolio-page.html](https://github.com/user-attachments/files/28724737/tailwindcss-portfolio-page.html) - Portfolio Page
'[tailwindcss-contact.html](https://github.com/user-attachments/files/28724735/tailwindcss-contact.html) - Contact page

All pages are connected through a shared navigation system, creating a seamless multi-page user experience.

Project Milestones

Milestone 1 – Project Setup

Completed the initial website structure by:

Creating three separate HTML files:
index.html
projects.html
contact.html
Linking the Tailwind CSS CDN within the <head> section of each page.
Setting up a <style type="text/tailwindcss"> block.
Defining custom colours using the @theme syntax.
Creating navigation links using <a> tags to connect all pages together.
Establishing a consistent project structure for future development.

Milestone 2 – Home Page Development

Focused on building the landing page experience by:

Creating a full-screen hero section.
Using CSS Grid to recreate the asymmetric layout from the wireframes.
Implementing large-scale typography to establish visual hierarchy.
Designing a Statistics section using a four-column layout.
Ensuring clean border alignment between statistics cards.
Building a Current Projects section.
Refining typography spacing and vertical alignment.
Applying responsive design principles to maintain usability across different screen sizes.

Milestone 3 – Projects Gallery

Developed the projects showcase page by:

Recreating the Selected Works archive layout.
Building responsive project cards.
Using Tailwind's responsive grid utilities:
grid-cols-1 md:grid-cols-2 lg:grid-cols-3
Creating a scalable gallery system that adapts to mobile, tablet, and desktop screens.
Maintaining visual consistency with the homepage design.

Milestone 4 – Contact Page

Recreated the "Leave Your Mark" contact page by:

Building a fully styled contact form.
Using Flexbox to align labels and form fields.
Creating a structured information panel.
Using vertical Flexbox layouts with consistent spacing through Tailwind's gap utilities.
Ensuring clean alignment and readability throughout the page.

Milestone 5 – User Experience & Polish

Focused on improving the overall user experience by:

Standardising margins and padding throughout the website.
Creating consistent spacing systems.
Adding hover effects to interactive elements.
Applying:
transition-colors
transition-transform
hover animations
Enhancing navigation responsiveness.
Creating a more polished and professional user experience.
